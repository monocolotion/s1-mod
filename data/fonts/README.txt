Place a CJK TrueType font file here to enable Chinese text rendering.

Supported filenames:
  - cjk.ttf (recommended)
  - cjk.otf

If no font is found here, the mod will try to use system fonts:
  - Microsoft YaHei (C:\Windows\Fonts\msyh.ttc)
  - SimSun (C:\Windows\Fonts\simsun.ttc)
  - SimHei (C:\Windows\Fonts\simhei.ttf)

Recommended free CJK fonts:
  - Noto Sans SC: https://fonts.google.com/noto/specimen/Noto+Sans+SC
  - Source Han Sans SC: https://github.com/adobe-fonts/source-han-sans
